Erlang Project Skeleton
=====
Install rebar3 by following the [Getting started guide](https://rebar3.readme.io/docs/getting-started)

Get dependencies
-----

    $ rebar3 get-deps
Build
-----

    $ rebar3 compile
Run
-----

    $ rebar3 shell

Make sure health is ok
-----
[http://localhost:8080/health](http://localhost:8080/health)
